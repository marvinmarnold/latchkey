What was built

  • Billing units fixed — all prices in dollars per million tokens, cost_usd in dollars. No more ambiguous micro-units.
  • LatchkeyBilling.sol — pull-payment contract. Callers approve once, proxy pulls when debt crosses threshold. 1% fee. onlyProxy guard.
  • Crash-safe pull worker — sign → persist (snapshot + hash + raw) → broadcast. Re-broadcasts saved raw tx on restart (idempotent, same hash).
  • RPC-free hot path — per-request gate reads local SQLite only. Allowance checked once for first-seen wallets, then cached.
  • Admin wallet table — GET /admin/wallets + live allowance reads. Dashboard shows accrued debt, lifetime pulled, allowance remaining, blocked status.
  • Ponder indexer scaffold — packages/indexer/ ready to index Pulled events once an Alchemy key is wired.

Remaining for Ponder live indexing

Set INDEXER_RPC_URL to an Alchemy Base Sepolia key in packages/indexer/.env (copy from .env.example), set INDEXER_START_BLOCK=42257000 (just before the contract deploy block), then cd packages/indexer && npm run dev. The admin dashboard will then show on-chain pull history in the WalletTable once the Ponder GraphQL query is wired.
